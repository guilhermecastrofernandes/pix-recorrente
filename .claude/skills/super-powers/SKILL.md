# Super Powers

Quando acionado para refatoração ou criação de novos módulos:
1. Aplique boas práticas de Clean Code e princípios SOLID em Java.
2. Sempre garanta cobertura de testes unitários com JUnit 5 e Mockito.
3. Analise o impacto das mudanças nas demais camadas da aplicação antes de salvar os arquivos.
